talk

Reinforcement Learning as a valid solution for real problems

Eduardo Pereira

Lead Specialist of Consulting Technology @ Delloite

about the talk

Reinforcement learning (RL) is a machine learning framework that address the optimization of decision planning processes. The value of RL have been shown primarily in video game environments, but its recent and consecutive breakthroughs have led to being applied and tested on various business scenarios. This talk will discuss the potential of this kind of approach.

workshop

Convolutional Neural Networks and Image Classification

João Matias

Data Scientist @ Talkdesk

about the talk

In this workshop we will have a brief introduction to deep learning and see some applications to image classification using Convolutional Neural Networks. Together, we will build an easy model to read numerical digits, using Python and Tensorflow, and see how the same tools can be applied to identify your favorite cartoon character in images.
